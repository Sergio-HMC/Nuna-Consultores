# TF Talent — Landing con Next.js

Proyecto listo para subir a GitHub y desplegar en Vercel.

## 1. Instalar dependencias

```bash
npm install
```

## 2. Correr en local

```bash
npm run dev
```

## 3. Deploy en Vercel

- Subir esta carpeta a un repositorio en GitHub
- Importar el repositorio en Vercel
- Deploy automático

## Estructura

- `app/page.tsx`: contenido de la landing
- `app/globals.css`: estilos globales
- `public/logo.svg`: logo
- `public/favicon.svg`: favicon

## Próximos ajustes recomendados

- Cambiar correo de contacto
- Añadir formulario conectado a Resend, Formspree o email
- Añadir Google Analytics o Meta Pixel si lo necesitan
- Reemplazar logo si definen una versión final de marca


## Nota de compatibilidad

Este proyecto fue actualizado a `next@15.2.8` para evitar versiones vulnerables bloqueadas por Vercel.


## Cambios de esta versión

- Logo actualizado a estilo manuscrito/cursivo
- Mayor protagonismo del muted brown
